Kuro Gboun Personal Portfolio Website

This is the personal portfolio website of Kuro Gboun, a 3rd-year Computer Science student specializing in Software Engineering. The website showcases Kuro's projects, skills, and contact information.

Author:
Kuro Gboun

Technologies Used:
HTML & CSS: For the structure and styling of the website.
JavaScript: For interactivity and dynamic content.
Images: Various icons and profile pictures used throughout the site.

Features:
Responsive Navigation: A desktop and mobile-friendly navigation system, including a hamburger menu for smaller screens.
Profile Section: An introduction to Kuro Gboun with a downloadable CV option and contact info.
About Section: Detailed information about Kuro's education and interests.
Projects Section: Showcase of completed projects with descriptions and links to relevant GitHub repositories.
Contact Section: Contact information, including phone number, email, LinkedIn, and GitHub.

Installation Instructions:
Clone or Download the Project:
